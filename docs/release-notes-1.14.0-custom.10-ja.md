# Spaced Repetition (Custom) 1.14.0-custom.10

## 変更内容

- 旧 plugin ID `obsidian-spaced-repetition:*` を参照する既存の Note Toolbar / Hotkeys 設定でもコマンドが動く互換レイヤーを追加
- Vault 起動時に `hotkeys.json` と `note-toolbar/data.json` の旧 command ID を新 ID へ自動移行
- 互換エイリアス登録と設定移行を検証する単体テストを追加

このプラグインは BRAT 経由でインストールまたは更新してください。
