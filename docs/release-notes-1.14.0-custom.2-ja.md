# Spaced Repetition (Custom) 1.14.0-custom.2

## 概要

フラッシュカード画面で、元ノート情報の表示位置と表示内容を見直した更新です。

## 主な変更

- 元ノート情報をプログレスバー直下から、`解答を表示` ボタンの下へ移動
- 質問文の視認性を上げるため、カード本文の上部をすっきり整理
- ノート情報の末尾に出ていた `Summary` 表示を除外

## インストール

このプラグインは BRAT 経由でインストールまたは更新してください。

`https://github.com/tk-pkm111/obsidian-spaced-repetition-custom`
