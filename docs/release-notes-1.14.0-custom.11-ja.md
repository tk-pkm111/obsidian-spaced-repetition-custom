# Spaced Repetition (Custom) 1.14.0-custom.11

## 変更内容

- レビューキュー（右サイドバー）の固定リスケボタン（1日 / 3日 / 7日）を廃止し、正規の間隔反復アルゴリズムを通すボタンに変更
- ボタンを「短く（Hard）/ スキップ（Good）/ 長く（Easy）」に変更。幅の都合で表示は漢字1文字（短 / 飛 / 長）、ホバーで日本語フルラベルをツールチップ表示
- 固定間隔へのリスケをやめ、ease に応じて復習間隔が本来どおり伸びるようになり、反復学習の効果が得られるよう修正
- エディタ右クリックメニューのコールバック引数に型注釈を追加（型チェック・Lint のクリーン化）

このプラグインは BRAT 経由でインストールまたは更新してください。
