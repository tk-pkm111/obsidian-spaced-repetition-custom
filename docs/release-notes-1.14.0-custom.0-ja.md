# Spaced Repetition (Custom) 1.14.0-custom.0

## 概要

Obsidian Spaced Repetition のカスタム fork の初回公開版です。

## 主な変更

- レビュー画面とデッキ画面の UI を全面的に改善
- フローティングバーによる最小化 / 復帰を追加
- 元ノートへ直接移動できる導線を追加
- 画像 / Excalidraw 埋め込みの表示と拡大の挙動を改善
- 右クリックからフラッシュカードを作成する導線を追加
- `#cornell` と `>[!cue]` 記法を使った Cornell カードに対応
- ナビゲーションと復帰時の操作不具合を複数修正

## インストール

このプラグインは BRAT 経由でインストールしてください。

リポジトリ:

`https://github.com/tk-pkm111/obsidian-spaced-repetition-custom`
